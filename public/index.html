Ex. 1 - Filter by Geographic Bounds

Show Features within an Area

This example shows a data service that is filtered by the point furthest north, east, south, and west. This limits the response and reduces the overall size of the request, making displaying subsets of large datasets more efficient

Ex 2 - Dynamically Filter

Using a slider calculated from the data, filter crimes.

This example shows a data service that is filtered by extend of the map, as well as by a date range chosen on a slider and by a weapon dropdown based on weapons listed in the data.

Go to Example

Ex. 3 - Filter & Buffer

Show Features within an Area, filtered, and buffered

This example shows a data service that if filtered by map extent, user input date slider and weapon dropdowns, and allows the user to input a buffer distance. Buffers are projected to EPSG 26985, NAD83/State Plane MD and then reprojected to WGS84 for display on the map.
